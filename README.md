
# AHA HOLISTIC – Gratis GitHub-hemsida

## Publicera gratis på GitHub Pages

1. Skapa konto på https://github.com
2. Klicka **New repository**
3. Döp det till `aha-holistic`
4. Klicka **Create repository**
5. Ladda upp filerna `index.html` och `style.css`
6. Gå till **Settings → Pages**
7. Under **Source** välj **Deploy from a branch**
8. Välj **main** och **/root**
9. Klicka **Save**

Efter cirka 1 minut blir sidan live på:

https://DITT-NAMN.github.io/aha-holistic/

Du kan senare koppla domänen `ahaholistic.se` till samma sida.
